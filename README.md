# JWT-Django
Authentication using django framework. (⚙️)

## Steps:
`>/ cd app`
`>/ python -m venv venv`
`>/ .\venv\Scripts\activate`

`>/ pip install -r requirements.txt`
`>/ pipenv install django`

## requirements.txt:
pipenv
django-seed
bcrypt
djangorestframework-simplejwt
djangorestframework
django-cors-headers
psycopg2
python-decouple
django-extensions
